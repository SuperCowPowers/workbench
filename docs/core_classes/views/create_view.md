# CreateView

!!! warning inline end "Experimental"
    The Workbench View classes are currently in experimental mode so have fun but expect issues and API changes going forward.

`CreateView` is the abstract base class for view creation (training, display, computation, etc.). The concrete view builders — [ColumnSubsetView](column_subset_view.md), [PandasToView](pandas_to_view.md), [ComputationView](computation_view.md), and others — subclass it.

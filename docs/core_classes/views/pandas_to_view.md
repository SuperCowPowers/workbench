# PandasToView

!!! warning inline end "Experimental"
    The Workbench View classes are currently in experimental mode so have fun but expect issues and API changes going forward.

`PandasToView` creates a view by joining an artifact's source table with a Pandas DataFrame on an id column — handy for layering feature-engineered columns onto an existing DataSource or FeatureSet. It subclasses [CreateView](create_view.md).
